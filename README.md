# DSA Solutions

My solutions to Data Structures & Algorithms problems — primarily from LeetCode — written in Python. This is an active, growing collection as I work through topics systematically.

## Topics Covered

| Topic | Examples |
|-------|---------|
| Arrays & Strings | Sliding window, two pointers, prefix sums |
| Linked Lists | Traversal, reversal, cycle detection |
| Trees & Graphs | BFS, DFS, tree traversal |
| Sorting & Searching | Binary search, merge sort |
| Dynamic Programming | Memoisation, tabulation |
| Bit Manipulation | XOR tricks, bit masking |

## Structure

Each solution file is named after the problem for easy searching. Solutions include comments explaining the approach and time/space complexity where relevant.

## Why Python?

Python lets me focus on the algorithm rather than syntax overhead, making it ideal for practicing and understanding problem-solving patterns.

## How to Run

```bash
# Clone the repo
git clone https://github.com/YakVash/DSA-Solutions.git
cd DSA-Solutions

# Run any solution
python solution_name.py
```
